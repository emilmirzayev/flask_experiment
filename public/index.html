Welcome to this experiment! We appreciate your participation.

The experiment constitutes a part of the academic research project studying the usage of recommender systems in search environments. Before you start the main part of the experiment, make sure you read instructions carefully. You will not be able to go back after you start. Refreshing the browser will cause the data to be lost. Please remember that after completing the task, you will have to copy the survey code given to you and paste it into Mturk to get compensated. The experiment consists of two stages: a choice task and a questionnaire. Your compensation depends on your performance in the choice task. It is not affected by your answers in your questionnaire. However, you need to complete the questionnaire to qualify for the payment.

In the choice task you will be given a table with one hundred items, each characterized by five features (X, Y, …, V). All features are numeric. Your task is to find and select 5 items with highest possible value of X + Z + V (i.e., the sum of the numbers associated to each of the three features). You have 10 minutes to complete the task. The task will automatically terminate at the end of 10 minutes. Make sure to make the choice (select and submit 5 items) before time runs out to qualify for payment. Once you have selected 5 options and would like to terminate the task, you need to confirm your choices to move to the post-task questionnaire.

For performing the task, you can potentially interact with the table in two ways: 1) you can hide features in the table (using hide feature button), and/or 2) you can sort items in increasing/decreasing order based on a given feature (with arrows next to each feature). Note that, given the aim of the experiment, you will be randomly allocated to one of the scenarios which will or will not allow you to use one or both of those interaction tools. To assist you in the search task, you will be provided with a set of recommended options. The recommendation algorithm is not aware of the precise aim of your task and considers all features when generating recommendations. Based on your behavior during the search process, the recommender system might revise recommended set to better help you in the search process. Note that you can select and de-select items multiple times. You can select items either from the main search table or from the table with recommended options.
